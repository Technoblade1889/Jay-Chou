# Jay-Chou页面编后感
## 关于思路来源
   从小时候就一直听jay一直听到现在，对于他的喜爱不止是他的歌，也还有他专辑的艺术风格，人生经历等等等。之前就有想过，我能不能自己创一个东西来收集所有我想了解的、未了解的来给我自己看，也给所有同好看。如今接触到了前端，就正好有机会得以完成我这个愿望。

## 关于编辑网站的学习过程
   万事开头难。刚开始接触到HTML与CSS的时候不亚于听天书，看网课也是啥都看不懂，只能勉强记住那几个关键词还有必要的结构。当时看的网课很多很杂，很多知识我根本记不住，当时也不知道该咋办，是继续这样学还是怎么样。不仅仅是我，我宿舍的也是如此，都觉得学的东西太多不懂怎么消化。后面经过争论经过交流，我们还是找到适合于我们的办法。在学习的时候就直接把学到的打在VS code的代码里，在指令的后面写注释当作笔记，上到新的就开新的网页，这样就渐渐完善了我们的笔记，如果真的太多导致忘了那就回去翻到那里就好；然后在我的建议下，我们也不再纯听网课，而是带他们一起想好主题，然后直接开始上手做网页。在我看来，跟网课学的再多笔记记得再好，随着时间的推移也总是会忘，那不如直接把学到的用于创建自己的网站，学到哪创到哪，又不会的就问ai，如果ai答到的东西根本没听过就继续看网课，看懂了再继续写。就如此循环往复。得谢于我们宿舍的学习方法和这种氛围，我们都完成的很快，早在16号我们就完成的大部分，后面的几天都是在完善，而其他宿舍的甚至才刚刚开始。

## 关于对html和CSS的感悟
   这一阶段的考核任务是我觉得最有意思的，前端敲的代码是可以直接看到回馈的，每敲完一段我就会打开网页来看，那种逐渐完成的感觉很舒服。html就是骨架，而css就是皮肤装修，说不上谁比谁重要，但我个人觉得写css的过程更有意思：去改变整体布局、改变画面艺术风格、改变字体形式......这些得到的反馈非常强烈。

## 关于这个网站的计划
   目前我也只是做了个主页，相应的活动、超链接等都没有完善，目前的内容也只是简简单单的个人介绍。等后续时间我想将每一张专辑都放到页面里，点击即可跳转到它的介绍。各种细节还有待完善。
